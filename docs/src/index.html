Whiteboard is slideshow application for demonstration-heavy presentations. It eliminates the need to be switching between code, browser, terminals, and descriptive text during live demonstrations, and enables more content to be prepared ahead of time. It's great for guiding code-heavy classroom lectures, tutorials, or workshops, or webinars.
